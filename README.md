# Personal_Notes_App
The Personal Notes App allows users to securely create, manage, and organize their personal notes with role-based access control.

## User Roles
- **Regular User:** Can create, view, edit, and delete only their own notes
- **Admin:** Can manage all users’ notes and user accounts

## Core Pages
- Login  
- User Dashboard  
- Notes List  
- Add / Edit Note  
- View / Delete Note  

## CRUD Operations
- **Create:** Add new notes  
- **Read:** View notes  
- **Update:** Edit existing notes  
- **Delete:** Remove notes  

## Key Features
- Role-based access control (User & Admin)
- Secure note management
- Search and filter notes
- Categorize notes using tags
- Pin important notes for quick access

# Objective
To provide a simple and secure platform for managing personal notes while maintaining strict access control and ease of use.
